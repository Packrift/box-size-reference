@packrift/box-size-reference

Corrugated box sizing helpers for practical packaging workflows.

A dependency-free JavaScript reference package for normalizing box dimensions, parsing common size labels, measuring parcel girth, checking product fit, and ranking boxes by unused volume.

GitHub repository Packrift box size calculator Corrugated boxes guide

Install

npm install @packrift/box-size-reference

What it includes

Dimension normalization

Convert inches and centimeters while preserving length x width x height order.

Fit checks

Check if a product fits inside a box with configurable clearance and rotation.

Parcel measurements

Calculate longest side, girth, and length plus girth for carrier screening.

Box recommendations

Rank fitting corrugated boxes by the lowest unused volume.

Example

import {
  fitsProduct,
  parseBoxSize,
  recommendBoxSizes,
  volume
} from "@packrift/box-size-reference";

const box = parseBoxSize("12 x 9 x 6 in");
const product = { length: 10, width: 5, height: 2 };

console.log(volume(box));
// { value: 648, unit: "in3" }

console.log(fitsProduct(product, box, { clearance: 0.5 }));
// true

const ranked = recommendBoxSizes(product, [
  { id: "10x6x4", length: 10, width: 6, height: 4 },
  { id: "12x9x6", length: 12, width: 9, height: 6 },
  { id: "14x10x8", length: 14, width: 10, height: 8 }
], { clearance: 0.5 });

console.log(ranked.map((item) => item.id));
// ["12x9x6", "14x10x8"]

API reference

Function Purpose
normalizeDimensions() Returns positive numeric dimensions in inches or centimeters.
parseBoxSize() Parses labels such as 12 x 9 x 6 in or 30*20*10 cm.
volume() Calculates cubic volume with a matching unit label.
parcelMeasurements() Returns longest side, width, height, girth, and length plus girth.
fitsProduct() Checks product fit with configurable clearance and rotation-aware side matching.
recommendBoxSizes() Filters fitting boxes and sorts them by lowest unused volume.
estimateOutsideDimensions() Estimates outside dimensions from inside dimensions, board thickness, and board thickness unit.

Reference notes

Corrugated boxes are commonly listed by inside dimensions in length x width x height order. Carrier rating and acceptance can depend on outside dimensions, dimensional weight, maximum length, and length-plus-girth rules. This package handles size math only; use current carrier rules and packaging tests for final shipping decisions.

For interactive guidance, see Packrift's box size calculator and corrugated boxes guide.
